# Phase 2: Processing

## What it does
Cleans and normalizes the raw reviews fetched in Phase 1. 
- Removes empty/null reviews.
- Trims whitespace.
- Redacts PII (Emails, Phone numbers).
- Removes duplicate reviews.

## Input File
- `data/raw_reviews.json`: Generated by Phase 1.

## Output File
- `data/processed_reviews.json`: The cleaned and deduplicated review dataset.

## How to run
```bash
node phase2_processing/run.js
```
