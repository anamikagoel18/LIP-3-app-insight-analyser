# Phase 3: Analysis

## What it does
Generates basic insights from the processed review data.
- Sentiment analysis based on star ratings.
- Keyword frequency extraction.
- Weekly aggregation.

## Input File
- `data/processed_reviews.json`: Generated by Phase 2.

## Output File
- `reports/weekly_report.json`: The final analysis summary.

## How to run
```bash
node phase3_analysis/run.js
```
